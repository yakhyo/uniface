Blazing-fast face analysis on Apple Silicon with native MLX acceleration

Why MLX-UniFace?

⚡

Native MLX Acceleration

Built from the ground up for Apple Silicon, leveraging unified memory architecture for zero-copy GPU operations.

🧠

Complete Face Analysis

Detection, recognition, 106-point landmarks, age, gender, and emotion analysis - all in one library.

🔄

Dual Backend Support

Automatic fallback to ONNX Runtime on non-Apple platforms. Write once, run anywhere.

✅

Numerical Parity

MLX outputs match ONNX with correlation > 0.999. Same accuracy, faster execution.

🎯

Multiple Model Options

RetinaFace, SCRFD, YOLOv5Face for detection. ArcFace, MobileFace, SphereFace for recognition.

📦

Easy Installation

Simple pip install with automatic weight downloading. Get started in seconds.

MLX vs ONNX Comparison

Feature MLX Backend ONNX Backend
Apple Silicon Native ✓ Yes ✗ Via CoreML
Unified Memory ✓ Zero-copy GPU ✗ CPU-GPU transfer
Lazy Evaluation ✓ Graph optimization ✗ Eager execution
M1/M2/M3/M4 Optimized ✓ Full optimization Partial
Cross-Platform macOS only ✓ All platforms
CUDA Support ✗ Not applicable ✓ Yes

Get Started in Seconds

# Install from PyPI
pip install mlx-uniface[mlx]
